### Lessons Learned
- **PostgREST cross-schema embed** — resolves only within from-clause schema. Fix is a same-schema VIEW proxying the cross-schema target. Saved hours today; saved as memory.
- **DB denormalization beats consumer refactor** — when an embed-shape fix would touch 5+ files, add denormalized column + sync trigger. Saved 13-file refactor today; saved as memory.
- **Verify brief scope before strategy** — briefs can be off by 100× on scale. Run 30-second REST count before accepting Director-locked tooling. Caught 65→4 chapters error today; saved as memory.
- **24-agent relentless decomposition pattern works** — non-overlapping file/cluster ownership, single named commit per agent, parallel worktrees. Fixed ~110 bugs in 5 batches without conflicts.
- **httpOnly cookie session injection** — `document.cookie` from JS can't set httpOnly cookies; production use must go through server actions like /yi-future/unlock. CFT-test the real flow, don't fake it.
- **Wrong cwd in worktree gotcha** — after sourcing files from absolute paths, bash relative paths inside agents' worktree dirs went to wrong location. Always verify pwd before `.env.local` reads in agent contexts.
- **Auto-save hook can swing HEAD between commits** — branch changed twice mid-session-end commit. Cherry-pick recovers cleanly.
