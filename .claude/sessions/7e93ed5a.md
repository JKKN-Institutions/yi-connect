### Lessons Learned

- **CFT can complete gated GCP admin actions via owned resources.** When director@ couldn't edit the OLD OAuth client, creating a fresh GCP project they own + recreating the client took 15 minutes vs unknown days waiting for escalation. Pattern: "user X can't do Y on resource A" → ask "can user X do Y on a fresh resource A' they own?" first. Saved to `feedback_cft_owned_resources_unblock_gated`.
- **JS .click() no-ops on Radix Tabs / Material Design / Google Cloud Console commit buttons.** Cost ~15 min total this session (Email Link tab false-positive + Google OAuth Create button). Use CDP Input.dispatchMouseEvent or ask user for real mouse click. Saved to `feedback_synthetic_click_misses_modern_ui_libs`.
- **Phase A schema ports copy RLS predicates verbatim, creating recursion class.** Predicate `chapter_id IN (SELECT m.chapter_id FROM yi_connect.members m WHERE m.id = auth.uid())` recurses when members' own policy uses the same pattern. Fix: subquery `yi_connect.profiles` (non-recursive RLS) instead. Saved to `feedback_rls_recursion_pattern`.
- **Code/DB column drift after schema migration is a distinct bug class.** Three flavors hit this session: rename mismatch (fiscal_year/calendar_year), missing column (skill_will_category), missing FK in PostgREST cache (events.organizer_id). All surfaced as `Failed to fetch X` with buried real error. Saved to `feedback_phase_d_column_drift`.
- **Parallel worktree agents (3-4 per batch on non-overlapping file sets) scale well.** Ran 7 agents across 2 batches + 4 agents in a follow-up — all completed cleanly. Pattern: one agent per failing route OR per concern (RLS / middleware / observability / data) with explicit file allowlists.
