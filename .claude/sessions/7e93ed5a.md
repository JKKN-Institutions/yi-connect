### Lessons Learned

- **Phase D's "190+282 files" report was incomplete.** Subagents that ported pages/components didn't sweep `src/app/actions/`. Always cross-check directory tree of source vs target after a port — don't trust file counts alone.
- **React 19 auto-hoists `<link>`/`<meta>` from any layer.** Splash-image links in nested layouts work without explicit `<head>` access. Cleaner than juggling metadata.other or extending root layout.
- **Two parallel UI primitive families is fine.** yi-connect (Radix shadcn) + YIP (base-ui shadcn) coexist by namespace. No need to consolidate.
- **Supabase TS generics pin schema literally.** `createServerClient<Database>` defaults to `"public"`; for non-public schemas, pass the schema name as the second generic.
- **macOS `rm -rf` is permission-gated even on freshly-created dirs.** Use `find ... -delete && rmdir` instead.
